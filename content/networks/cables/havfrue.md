## Overview

**Havfrue** (Danish for "mermaid") is a transatlantic submarine cable system connecting the United States to Denmark, with branches to Ireland and Norway. The Norwegian branch is wholly owned by **Bulk Infrastructure**.

| Attribute | Value |
|-----------|-------|
| **Status** | ✅ Active |
| **Ready for Service** | 2019 |
| **Total Length** | 7,880 km |
| **Norwegian Fiber Pairs** | 2 |
| **Norwegian Owner** | Bulk Infrastructure |

## Route

```
Wall Township, NJ (USA) → Blaabjerg (Denmark)
                       ├─→ Killala (Ireland)
                       └─→ Kristiansand (Norway)
```

## Consortium

The Havfrue consortium includes:
- **Aqua Comms** - System operator, 2 fiber pairs on main trunk
- **Meta (Facebook)** - 2 fiber pairs on main trunk
- **Google** - 1 fiber pair on main trunk
- **Bulk Infrastructure** - 1 fiber pair on main trunk + **all 2 fiber pairs on Norway branch**

## Norwegian Significance

The Norwegian branch provides:
- Direct transatlantic connectivity for Norwegian datacenters
- Low-latency route to US markets
- Strategic importance for Nordic digital infrastructure

The cable lands at **Kristiansand**, connecting to Bulk Infrastructure's fiber networks and datacenter facilities.
