Eliminate 'it works on my machine' issues with a standardized, containerized development setup. Pre-configured with Azure CLI, Python, Node.js, and essential tools.

## Features

- Pre-configured development environment using Debian 12 Bookworm
- Essential tools: Azure CLI, Python, Node.js, common CLI utilities
- VS Code extensions for Azure, PowerShell, Markdown, YAML
- Extensible architecture for role-specific tools
- Works with VS Code, JetBrains Rider, and Visual Studio

## Problems Solved

- Eliminates environment inconsistencies across team members
- Simplifies onboarding with ready-to-use setup
- Safe experimentation without affecting local machine
- Modular approach for adding development tools

## Template Library

Includes templates for web applications in C#, Python, Go, Java, PHP, TypeScript with frameworks like Next.js, React, Spring Boot, and Express.
